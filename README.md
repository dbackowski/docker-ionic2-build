# docker-ionic2-build

Dockerfile based on official node:6.10.0 docker image with android-sdk-linux for android builds.

## What is included?

* Node.js - 6.10.0
* Ionic framework - 2.2.0
* Cordova - 6.5.0
* Android SDK - 25.2.3

## Usage 

```
docker run damianbackowski/docker-ionic2-build [cmd]
```
